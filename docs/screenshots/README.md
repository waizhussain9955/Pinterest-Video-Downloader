## Adding Screenshots

To add screenshots:

1. Take high-quality images of the application
2. Save as PNG format for best quality
3. Name consistently (e.g., `feature-name.png`)
4. Update this README with new screenshots

## Screenshot Guidelines

- Include the full viewport
- Show key features clearly
- Use consistent styling
- Maintain aspect ratio
- Optimize file sizes
